##Korean/Japanese subtitles Dictionary Using Python + Postgresql
#Python基盤の日本語、韓国語字幕を利用する辞書

##機能
 * 日本語、韓国語字幕を自動的にデータベースに登録
 * ユーザーが入力したキーワードをデータベースと比べて両方の言語の検索結果を提供

#Enviroment & Requirement
 * Python 3.4.3
 * PostgreSQL 
 * psycopg2
